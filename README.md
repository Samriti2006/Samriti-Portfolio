 Samriti's Portfolio Website

A personal portfolio website built with HTML, CSS, and JavaScript 

🔗 Live site: https://samriti2006.github.io/Samriti-Portfolio/

## Features
- Dark/light mode toggle
- Typing animation
- Scroll-spy navigation with active section highlighting
- Filterable project section
- Certificate gallery with click-to-view lightbox
- Contact form powered by EmailJS

## Tech Stack
HTML5 · CSS3 · JavaScript (Vanilla) · EmailJS
